# Inception

> **This activity has been created as part of the 42 curriculum by mowaidat.**

## 📖 About This Repository

This repository contains my implementation of the **Inception** project from the **42 School** curriculum.

The purpose of this repository is not only to present the final project, but also to provide a complete learning guide for Docker and the technologies used throughout the project.

Every concept is explained from first principles using simple English, real-world examples, terminal commands, diagrams, and deep technical explanations.

Whether you are a 42 student or someone learning Docker for the first time, this repository is designed to help you understand **how everything works**, not just how to make it work.
